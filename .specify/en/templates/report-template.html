Root Cause Analysis

Root Cause

The database connection pool's maximum connection limit (50) was insufficient to handle burst traffic. The newly launched reporting feature introduced multiple long-running queries that occupied a large number of connections, preventing regular API requests from obtaining database connections.

The connection pool monitoring alert threshold was set too high, failing to provide early warning before connections were exhausted.
